**Telegram bot для проведения анкетирования**

* Можно подгружать вопросы из таблиц Excel
По одному вопросу в ячейке в первой колонке,
варианты ответов в следующих за вопросом колонках,
если нет варианта ответа - ответ вводит опрашиваемый

Текущие вопросы и ответы будут удалены!

* Результаты выгружаются в Ecxel формате.
